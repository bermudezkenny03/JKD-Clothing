# TailAdmin Pro - Vue (v2.0)

TailAdmin es una plantilla moderna y responsive para paneles administrativos, construida con **Vue 3, Tailwind CSS y Vite**. Ideal para crear dashboards profesionales de forma rápida.

## 🚀 Características Principales

- Diseños modernos y personalizables
- Dashboards listos para usar:
  - Ecommerce
  - Analytics
  - Marketing
  - CRM
- Sidebar colapsable
- Tablas avanzadas
- Calendario con drag & drop
- Chat en tiempo real
- Gráficos con ApexCharts
- Diseño responsive
- Buen rendimiento con Vue 3

## 📦 Requisitos

- Node.js 18 o superior (recomendado 20+)

## ⚡ Instalación

```bash
npm install
npm run dev
